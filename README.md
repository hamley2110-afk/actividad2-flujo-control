# Tarea 2 - Flujo de Control Parte 1
Este programa calcula el ISR de un empleado a partir de su sueldo mensual. Primero obtiene el sueldo anual y luego verifica si aplica el pago de impuestos según el rango correspondiente. Al final muestra el sueldo y el monto del ISR, o RD$0.00 si no corresponde pagar.

Funcionamiento

* Se ingresa el sueldo mensual.
* Se calcula el sueldo anual.
* Se compara con los rangos del ISR usando if y else if.
* Se muestra el resultado en pantalla.

Lenguaje utilizado

* C++
